# Rattle Revenue Dashboard

A modern revenue dashboard for tracking deal pipelines, metrics, and alerts. built a custom component library using TypeScript, React, and Tailwind CSS.

## Quick Start

**Prerequisites:** Node.js 18+, pnpm 10.21.0+

```bash
pnpm install
pnpm dev
```

Open `http://localhost:3000`

## Project Structure

```
src/
├── components/    # UI components and design system
├── routes/        # File-based routing (TanStack Router)
├── data/          # Mock data
├── types/         # TypeScript definitions
└── utils/         # Helpers and design tokens
```

## Tech Stack

- React 19
- TypeScript
- TanStack Router
- Tailwind CSS
- Vite
- Biome
